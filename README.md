# 小小侦探破案局：一年级两步应用题练习

面向小学一年级孩子的互动式数学练习网页。产品围绕“比谁多、比谁少”的连续两问应用题，不让孩子直接猜答案，而是按题目中的关系一步步拆解：找标准、找比较对象、拉线段、写关系式、再完成两个问题。

## 在线体验

- 访问地址：[https://carrot-gaooo.github.io/mathematics-V2/](https://carrot-gaooo.github.io/mathematics-V2/)
- 当前版本：`mathematics-V2`
- 使用方式：打开网页即可使用，不需要安装。

## 核心功能

| 模块 | 说明 |
|---|---|
| 自动出题 | 每轮自动生成 3 道连续两问应用题，题材覆盖动物园、公交站、文具店、果园、玩具角等生活场景。 |
| 难度选择 | 支持 `20以内`、`50以内`、`100以内` 三档加减法练习。 |
| 题目信息高亮 | 每道题可单独打开高亮，突出“比”前后对象、数量和多/少关系。 |
| 线段交互 | 在线段上直接拖动，持续显示当前数值；拖到正确位置后红字标注“正确”。 |
| 四步答题 | 按“大数/小数/差值关系式 -> 数字等式 -> 第一问答案 -> 第二问答案”完成推理。 |
| 错题库 | 自动收集做错的题，便于下一次集中复盘。 |
| 趣味视觉 | 题目区域配有对应场景图形和小动物元素，降低应用题阅读压力。 |

## 解题流程

1. 标出“比”后面的内容和数字，作为标准线段。
2. 标出“比”前面的内容和数字，作为第二条解题线段。
3. 第二条线段拆成两部分：绝对值部分和差值部分。
4. 写出大数、小数、差值之间的加减法关系。
5. 填入数字等式。
6. 分别回答第一问和第二问。

## 设计目标

- 让孩子从“算答案”转向“看懂数量关系”。
- 把抽象的“多/少/差值”变成可拖动、可观察的线段。
- 通过 3 题一轮、即时反馈、错题收集，形成短时高频练习闭环。

## 技术说明

- 静态单页 HTML 应用。
- 不依赖后端服务。
- 错题库保存在当前浏览器本地。
